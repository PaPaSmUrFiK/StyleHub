# Папки с изображениями одежды

Эта папка содержит изображения одежды, которые автоматически загружаются в конструктор образов.

## Структура папок

```
public/images/
├── headwear/  # Головные уборы
├── top/       # Верхняя одежда
├── bottom/    # Нижняя одежда
└── shoes/     # Обувь
```

## Поддерживаемые форматы

- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.svg`

## Как это работает

1. Поместите изображения в соответствующую папку категории
2. Изображения автоматически загружаются при запуске приложения
3. В конструкторе образов по умолчанию отображаются первые 3 изображения каждой категории + пустая карточка
4. Для просмотра всех остальных изображений нажмите кнопку "Еще варианты"

## Именование файлов

Имена файлов используются для генерации названий элементов одежды:
- `red-cap.jpg` → "Red Cap"
- `blue-jeans.png` → "Blue Jeans"

Рекомендуется использовать понятные имена файлов на английском языке.
